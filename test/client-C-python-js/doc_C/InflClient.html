Description

La classe InflClient est la classe la plus importante de cette bibliothèque. C'est cette classe qui permet de se connecter, d'attaquer, de placer ses unités et de passer au tour suivant. Une instance de InflClient doit se libérer avec g_object_unref() lorsqu'elle n'est plus nécessaire. Il est important de libérer les instances de cette classe puisque la déconnexion se fait lorsque la classe est libérée.

infl_client_get_my_cells ()

void
infl_client_get_my_cells (InflClient *client,
                          InflCell ***cells,
                          int *n);

Permet de récupérer les cellules appartenant à votre équipe. La fonction s'occupe d'allouer la mémoire du tableau contenant ces cellules. Le tableau retourné par cette méthode doit être libéré avec free().

Exemple d'utilisation:

// On suppose qu'un client est disponible dans la variable 'client'

InflClient **my_cells; // Un tableau de cellules
int n_cells; // Contiendra la taille de my_cells

infl_client_get_my_cells(client, &my_cells, &n_cells);

// Traitement...

free(my_cells) // on libère my_cells quand on en a plus besoin

[skip]

Parameters

client

Un client

  

cells

Un pointeur vers un tableau de InflCell *

  

n

Un pointeur qui stockera la taille du tableau.
