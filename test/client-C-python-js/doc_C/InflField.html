Description

La classe InflField gère la carte du jeu et permet de récupérer les cellules de cette carte. L'utilisateur n'a normalement pas besoin de créer des instances de cette classe. Les méthodes de la classe InflClient retournent une nouvelle carte aux moments où le plateau est modifié.
