Documentation du client

Le client est divisé en 3 classes. Vous utiliserez surtout la classe Infl.Client.

Les propriétés

Chaque classe contient une section "Properties". Il y a deux manières d'utiliser les propriétés listées dans ces sections. Par exemple, si l'instance s'appelle cell et la propriété s'appelle x, on peut soit utiliser les getters et les setters

v = cell.get_x()
cell.set_x(42)

soit utiliser l'attribut spécial props

v = cell.props.x
cell.props.x = 42

attention, certaines propriétés ne sont pas accessibles en écriture. Si vous essayez de changer leur valeur, vous obtiendrez une erreur.
