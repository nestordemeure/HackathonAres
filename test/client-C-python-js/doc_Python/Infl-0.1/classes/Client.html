add_unit_list(units)¶
Parameters:units ([Infl.Cell]) – Un tableau de cellules

Ajoute une unité pour chaque cellule listée dans units. Cette méthode est soumise aux mêmes contraintes d’utilisations que Infl.Client.add_units().

add_units(c, count)¶
Parameters:
  • c (Infl.Cell) – La cellules sur laquelle ajouter des unités
  • count (int) – Le nombre d’unités à ajouter

Ajoute des unités à une cellule. Cette méthode s’utilise uniquement pendant la phase de rechargement. La valeur de retour de Infl.Client.end_attacks() indique le nombre maximal d’unités disponibles pour le rechargement lors du tour.
