class Infl.ClientStatus(value)¶

Bases: GLib.Enum

Cette énumération est utilisée par la méthode Infl.Client.get_status() pour indiquer le statut du client. Le serveur ne réagira pas si le statut est différent de Infl.ClientStatus.ONGOING.

CONNECTION_LOST = 0¶

Problème de réseau.

ONGOING = 1¶

Partie en cours.

VICTORY = 2¶

Victoire du client.

DEFEAT = 3¶

Défaite du client.
