class Gio.EmblemedIcon(**kwargs)¶
Bases:GObject.Object, Gio.Icon
Abstract:No
Structure:Gio.EmblemedIconClass

Gio.EmblemedIcon is an implementation of Gio.Icon that supports adding an emblem to an icon. Adding multiple emblems to an icon is ensured via Gio.EmblemedIcon.add_emblem().

Note that Gio.EmblemedIcon allows no control over the position of the emblems. See also Gio.Emblem for more information.
